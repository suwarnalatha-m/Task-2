About the Event

IFET Coders is a one-day event to encourage cross-discipline collaboration. Teams can present software projects or hardware prototypes. Judges value working demos, clear problem statements, and scalability. Students with profiles on coding platforms (CodeChef, HackerRank, GitHub) are encouraged to mention them during registration.

How to Present (Quick Guide)

  • Describe the problem & target users in 30 seconds.
  • Show the working demo or screenshots; explain the core flow.
  • Highlight what you built and what’s left to scale.
  • If hardware involved, demonstrate the data pipeline & UI, not just the circuit.
